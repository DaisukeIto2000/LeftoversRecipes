## LeftoversRecipe(概要)

料理レシピを投稿や検索できるアプリケーションです。
ログインすると材料からレシピを検索できる、余り物レシピ検索を利用することができます。こちらは冷蔵庫などにある材料を入力することで該当するレシピを検索できます。
